# Claude 작업 원칙

## 필수 준수 사항

1. **시키지 않은 일은 하지 않는다**
   - 커밋, 푸시, 릴리즈 등 중요한 작업은 사용자가 명시적으로 요청한 경우에만 실행
   - 추측하거나 선제적으로 작업하지 않음

2. **Python은 사용하지 않는다**
   - 데이터 분석이나 처리가 필요할 때도 Bash나 다른 도구 사용
   - 절대 Python 스크립트를 작성하거나 실행하지 않음
   - 어쩔수 없이 필요한 경우에만 사용자에게 물어봄.

3. **특정한 값이 필요하면 직접 확인한다**
   - 사용자에게 값을 물어보지 않고 코드나 데이터를 직접 확인해서 찾기
   - 특히 만들어진 어플리케이션에서 생성하는 값은 직접 실행해서 확인하기
   - 그외에 필요한 정보는 파일 읽기, grep, 데이터 조회 등으로 직접 확인
